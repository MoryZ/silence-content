# 实体类注解增强指南

## 概述

本文档说明实体类生成器对特殊字段的注解支持，包括以`is`开头的boolean字段和Map类型的扩展属性字段。

## 功能特性

### 1. @Column 注解支持（is前缀字段）

#### 问题背景
JPA/Hibernate在处理以`is`开头的boolean字段时，会自动去掉`is`前缀生成getter/setter方法，导致字段映射错误。

例如：
- 数据库字段：`is_deleted`
- Java字段：`isDeleted`  
- Getter方法：`getDeleted()` (JPA自动去掉is前缀)
- 实际映射字段：`deleted` ❌ 错误！

#### 解决方案
对于以`is`开头的boolean字段，自动添加`@Column`注解显式指定数据库字段名：

```java
@Column(name = "is_deleted")
private Boolean deleted;
```

#### 检测规则
1. 字段类型必须是`Boolean`或`boolean`
2. 字段名以`is`开头
3. 第3个字符是大写字母（如`isDeleted`、`isStickyTop`）

#### 代码示例

**数据库表结构：**
```sql
CREATE TABLE content_article (
    id BIGINT PRIMARY KEY,
    title VARCHAR(200),
    is_deleted TINYINT(1) DEFAULT 0,
    is_sticky_top TINYINT(1) DEFAULT 0,
    created_at TIMESTAMP
);
```

**生成的实体类：**
```java
@Entity
public class ContentArticle extends AbstractAutoGeneratedIdAuditable<Long> {
    
    /**
     * 标题
     */
    private String title;
    
    /**
     * 是否删除
     */
    @Column(name = "is_deleted")
    private Boolean deleted;
    
    /**
     * 是否置顶
     */
    @Column(name = "is_sticky_top")
    private Boolean stickyTop;
    
    // getter/setter 省略...
}
```

### 2. @Convert 注解支持（Map扩展属性）

#### 问题背景
对于使用JSON格式存储的扩展属性字段（如`attributes`、`ext_attributes`），需要在实体类中映射为`Map<String, Object>`类型，并使用自定义转换器进行JSON和Java对象之间的转换。

#### 解决方案
对于Map类型的扩展属性字段，自动添加`@Convert`注解指定转换器：

```java
@Convert(converter = MapAttributeConverter.class)
private Map<String, Object> attributes;
```

#### 检测规则
1. 字段名为`attributes`或`extAttributes`
2. 字段类型为`Map<String, Object>`

#### 代码示例

**数据库表结构：**
```sql
CREATE TABLE content_article (
    id BIGINT PRIMARY KEY,
    title VARCHAR(200),
    attributes JSON,  -- 扩展属性
    created_at TIMESTAMP
);
```

**生成的实体类：**
```java
import jakarta.persistence.Convert;
import com.old.silence.data.jdbc.core.converter.MapAttributeConverter;
import java.util.Map;

@Entity
public class ContentArticle extends AbstractAutoGeneratedIdAuditable<Long> {
    
    /**
     * 标题
     */
    private String title;
    
    /**
     * 扩展属性
     */
    @Convert(converter = MapAttributeConverter.class)
    private Map<String, Object> attributes;
    
    // getter/setter 省略...
}
```

### 3. 完整示例

**数据库表：**
```sql
CREATE TABLE content_notice (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(200) NOT NULL COMMENT '标题',
    content TEXT COMMENT '内容',
    is_deleted TINYINT(1) DEFAULT 0 COMMENT '是否删除',
    is_published TINYINT(1) DEFAULT 0 COMMENT '是否发布',
    attributes JSON COMMENT '扩展属性',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

**生成的实体类：**
```java
package com.old.silence.content.model;

import jakarta.persistence.Column;
import jakarta.persistence.Entity;
import jakarta.persistence.Convert;
import com.old.silence.data.commons.domain.AbstractAutoGeneratedIdAuditable;
import com.old.silence.data.jdbc.core.converter.MapAttributeConverter;
import java.util.Map;

/**
 * 公告实体类
 *
 * @author moryzang
 */
@Entity
public class ContentNotice extends AbstractAutoGeneratedIdAuditable<Long> {
    private static final long serialVersionUID = 1L;

    /**
     * 标题
     */
    private String title;

    /**
     * 内容
     */
    private String content;

    /**
     * 是否删除
     */
    @Column(name = "is_deleted")
    private Boolean deleted;

    /**
     * 是否发布
     */
    @Column(name = "is_published")
    private Boolean published;

    /**
     * 扩展属性
     */
    @Convert(converter = MapAttributeConverter.class)
    private Map<String, Object> attributes;

    // Getter和Setter方法
    public String getTitle() {
        return this.title;
    }

    public void setTitle(String title) {
        this.title = title;
    }

    public String getContent() {
        return this.content;
    }

    public void setContent(String content) {
        this.content = content;
    }

    public Boolean getDeleted() {
        return this.deleted;
    }

    public void setDeleted(Boolean deleted) {
        this.deleted = deleted;
    }

    public Boolean getPublished() {
        return this.published;
    }

    public void setPublished(Boolean published) {
        this.published = published;
    }

    public Map<String, Object> getAttributes() {
        return this.attributes;
    }

    public void setAttributes(Map<String, Object> attributes) {
        this.attributes = attributes;
    }
}
```

## 实现细节

### 模板变量（FreeMarker）

在`model.ftl`模板中可以使用以下工具方法：

- `needsColumnAnnotation(column)` - 判断字段是否需要@Column注解
- `isMapField(column)` - 判断字段是否是Map扩展属性
- `hasMapType` - 判断表是否包含Map类型字段（用于条件导入）

### 示例模板代码

```ftl
<#-- 导入jakarta.persistence注解 -->
import jakarta.persistence.Column;
import jakarta.persistence.Entity;
<#if hasMapType>
import jakarta.persistence.Convert;
import com.old.silence.data.jdbc.core.converter.MapAttributeConverter;
import java.util.Map;
</#if>

<#-- 字段定义 -->
<#list columnInfos as column>
    <#if !column.primaryKey>
    /**
     * ${column.comment}
     */
    <#if needsColumnAnnotation(column)>
    @Column(name = "${column.originalName}")
    </#if>
    <#if isMapField(column)>
    @Convert(converter = MapAttributeConverter.class)
    </#if>
    private ${getJavaType(column)} ${toCamelCase(column.fieldName, false)};
    </#if>
</#list>
```

```xml
    <artifactId>silence-data-jdbc-core</artifactId>
</dependency>
```

## 注意事项

1. **is前缀字段**：只对boolean类型有效，其他类型不会添加@Column注解
2. **Map字段**：只对字段名为`attributes`或`extAttributes`的Map字段有效
3. **转换器**：`MapAttributeConverter`需要在项目中实现，用于处理JSON与Map的转换
4. **导入优化**：只有存在Map类型字段时才会导入相关类，避免无用导入
